Theme background images for the web app.

The background changes with the theme (Ion, Lava, Phoenix, etc.) like the desktop
version. Expected filenames: Ion.png, Lava.png, Phoenix.png, Violet.png,
Blue_Balls.png, Artic Hex.png, Carbon_Flux.png, Platinum.png.

If you have the desktop BL4 Save Editor, you can copy its BG_Themes/*.png here,
or run from repo root: npm run copy-bg-themes (from web folder: npm run copy-bg-themes).
If these files are missing, the app uses a solid dark background.
