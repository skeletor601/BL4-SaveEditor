Place your theme background images here. Each image filename must match a theme name (case-sensitive).

Supported themes and expected filenames:
  Ion         -> Ion.png or Ion.jpg
  Lava        -> Lava.png or Lava.jpg
  Phoenix     -> Phoenix.png or Phoenix.jpg
  Violet      -> Violet.png or Violet.jpg
  Blue_Balls  -> Blue_Balls.png or Blue_Balls.jpg
  Artic_Hex   -> Artic_Hex.png or Artic_Hex.jpg
  Carbon_Flux -> Carbon_Flux.png or Carbon_Flux.jpg
  Platinum    -> Platinum.png or Platinum.jpg

The app will use the first file found (.png then .jpg) for each theme. If a file is missing, a solid dark background is used for that theme.
